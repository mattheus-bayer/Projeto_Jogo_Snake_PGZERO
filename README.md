# Projeto_Jogo_Snake_PGZERO
Este projeto consiste em um jogo Snake desenvolvido em Python com a biblioteca PGZero. O jogador controla uma cobra verde que se movimenta continuamente pela tela, coletando maçãs vermelhas para aumentar a pontuação e o tamanho da cobra. O jogo utiliza lógica de programação, variáveis, movimentação por teclado e geração aleatória de elementos.
